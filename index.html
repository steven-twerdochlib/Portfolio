Description:
The aim of this tutorial was to learn how to create the basic functions necessary to make a simulator game using Roblox Studios. The basic functions involved in this simulator tutorial included; increasing a stat every time the user clicks, preventing the user from spam clicking (clicking multiple times within a given time frame), rebirthing (reseting certain stats but increasing the amount the stat increases by), changing the size of the players character based on certain stats, saving/loading data on exit and entering the game and displaying certain stats on a leaderboard to all players. This tutorial was completed by following three separate YouTube videos.
