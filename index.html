A Level Project
Slideshow Image: 1 / 4
Slideshow Image: 2 / 4
Slideshow Image: 3 / 4
Slideshow Image: 4 / 4
❮ ❯

Description:
The aim of this project was to create a game would be used to teach students the importance of patience in a fun/interesting way. The story of this gameplay of this project is as follows; the player starts off in a (partially) randomly generated maze and can only see the tiles around him, the user has to discover the entire maze to move on to the next level (undiscovered tiles are the grey tiles). The player can discover tiles by getting in range of the tiles. Throughout the maze their are enemies that on contact with the player ends the game, these enemies move in a random path. To make the game less RNG (about luck) there is a feature that allows the user to pass through walls by holding the spacebar key and pressing a move key simultaneously but once used there is a cooldown on when you can use it again. To make the game increasingly harder every level, every level completed the number of enemies in the maze increases by 1. This project also had documentation with it, however this has been lost between the time the project was done and the time this website portfolio was made. This project has some bugs and some features which have not been implemented yet like the save and load options but it is still a playable game which provides the origional intent of teaching students patience and my first game so I am very proud of it.

Roblox Simulator Tutorial
Slideshow Image: 1 / 3
Slideshow Image: 2 / 3
Slideshow Image: 3 / 3
❮ ❯

Description:
The aim of this tutorial was to learn how to create the basic functions necessary to make a simulator game using Roblox Studios. The basic functions involved in this simulator tutorial included; increasing a stat every time the user clicks, preventing the user from spam clicking (clicking multiple times within a given time frame), rebirthing (reseting certain stats but increasing the amount the stat increases by), changing the size of the players character based on certain stats, saving/loading data on exit and entering the game and displaying certain stats on a leaderboard to all players. This tutorial was completed by following three separate YouTube videos.
